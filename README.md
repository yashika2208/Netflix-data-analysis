Netflix Data Analysis

 📌 Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python. The analysis focuses on understanding Netflix's content library through data cleaning, exploration, and visualization.

 🎯 Objectives
- Clean and preprocess the dataset.
- Analyze movies and TV shows available on Netflix.
- Identify content trends over the years.
- Explore ratings, countries, and genres.
- Visualize key insights using charts.

 🛠️ Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

 📂 Dataset
The dataset used is the **Netflix Movies and TV Shows** dataset from Kaggle.

Dataset columns include:
- Show ID
- Type
- Title
- Director
- Cast
- Country
- Date Added
- Release Year
- Rating
- Duration
- Listed In (Genre)
- Description

 📊 Analysis Performed
- Data Cleaning
- Handling Missing Values
- Checking Duplicate Records
- Movies vs TV Shows Analysis
- Content Ratings Distribution
- Country-wise Content Analysis
- Release Year Analysis
- Top Genres Analysis

📈 Visualizations
- Distribution of Movies and TV Shows
- Content Rating Distribution
- Top 10 Countries with Most Content
- Netflix Content Added Over the Years
- Top Genres on Netflix

 🔍 Key Insights
- Movies make up a larger portion of Netflix's content than TV Shows.
- The United States has the highest number of titles.
- Most content was added between 2018 and 2020.
- TV-MA is one of the most common content ratings.
- Drama and International Movies are among the most popular genres.

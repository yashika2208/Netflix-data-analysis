Netflix Data Analysis

 Project Overview
This project analyzes the Netflix Movies and TV Shows dataset using Python, Pandas, Matplotlib, and Seaborn. The objective is to explore trends in Netflix content through data cleaning, analysis, and visualization.

 Dataset
- Source: Kaggle - Netflix Movies and TV Shows Dataset
- File Used: netflix_titles.csv

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

Analysis Performed
- Data cleaning
- Missing value handling
- Movies vs TV Shows comparison
- Content release year analysis
- Top producing countries
- Most common ratings
- Most popular genres

 Key Insights
- Movies make up most of Netflix's content.
- The United States and India contribute the highest number of titles.
- Netflix content increased significantly after 2015.
- TV-MA and TV-14 are the most common ratings.
- International Movies is the most popular genre.
